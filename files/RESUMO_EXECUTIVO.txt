📊 PERMISSÕES POR PERFIL
═════════════════════════════════════════════════════════════════════════════════

ADMIN_OM (Administrador):
  ├─ Gerencia 1 OM (sua OM principal)
  ├─ Cria/edita Escalas
  ├─ Publica Escalas
  ├─ Configura Calendário
  ├─ Adiciona Militares
  └─ Registra Indisponibilidades

ESCALANTE (Gerador):
  ├─ Cria Escalas
  ├─ Gera automaticamente
  ├─ Ajusta manualmente
  ├─ Visualiza Quadrinho
  └─ Não publica (admin faz)

MILITAR (Consulta):
  ├─ Visualiza sua escala
  ├─ Vê seus dias escalados
  ├─ Consulta indisponibilidades
  └─ Sem permissão de editar

GERENTE (Relatórios):
  ├─ Leitura de todas as escalas
  ├─ Gera relatórios
  ├─ Análise de balanceamento
  └─ Sem permissão de editar


⚙️ ALGORITMO DE GERAÇÃO AUTOMÁTICA
═════════════════════════════════════════════════════════════════════════════════

1. Obter todos os dias do mês (CalendarioDia)
2. Para cada dia:
   a. Consultar Quadrinho (ordenado por quantidade ASC)
   b. Verificar indisponibilidades
   c. Alocar primeiro militar disponível
   d. Signal incrementa Quadrinho automaticamente
3. Resultado: Escala balanceada em segundos

Benefício: Distribuição AUTOMÁTICA e EQUITATIVA de serviços


✅ VALIDAÇÕES IMPLEMENTADAS
═════════════════════════════════════════════════════════════════════════════════

Militar:
  ✓ CPF: 11 dígitos, não duplicado
  ✓ Idade: mínimo 18 anos
  ✓ Matrícula: UNIQUE por OM
  ✓ Divisão: deve pertencer à mesma OM

Escala:
  ✓ UNIQUE: (OM, tipo_escala, mes, ano)
  ✓ Status protege publicadas
  ✓ Mês entre 1-12
  ✓ Ano entre 2000-2100

EscalaItem:
  ✓ UNIQUE: (escala, dia) = 1 militar/dia
  ✓ Militar mesma OM que Escala
  ✓ Valida indisponibilidades
  ✓ Protege escalas publicadas

Indisponibilidade:
  ✓ Data fim >= Data início
  ✓ Máximo 365 dias
  ✓ Impede alocação automática

Quadrinho:
  ✓ UNIQUE: (militar, escala, serviço, ano)
  ✓ Nunca negativo
  ✓ Sincronizado com EscalaItem
